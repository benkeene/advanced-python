# Advanced Python: A Gentle Introduction to Python for Research Computing

1. Clone this repository
2. Set up your python environment and `pip install -r requirements.txt`
3. Run the setup script `./setup.sh`
4. Open the `workshop.ipynb` notebook and begin working.

*Note*: The LLM will be unavailable after 5pm 6/24/26. Configure your .env to point to another LLM if you'd like.
